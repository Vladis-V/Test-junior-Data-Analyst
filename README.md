# Тестовое задание на позицию Junior Data Analyst

В данном репозитории представлены решения тестового задания из 5 тематических блоков.

## Содержание:
1. **Теория вероятности и логика** (Ферма, Кулинарное соревнование, Одинокая дорога)
2. **Python** (Изоморфизмы, Натуральная последовательность, факторизация).
3. **SQL** (Абитуриенты, FULL JOIN, покупки).
4. **Statistics & AB-testing** (Воодушевленное руководство, Основной показатель в статистике, Параметрический тест).
5. **ML Base** (Пони тоже кони, Ручной счёт ROC_AUC, Ручной счёт корреляции).

## Используемый стек:
• **Python**
• **SQL** 
• **Математическая статистика и теория вероятностей**
